## 🚫 需要删除或不包括的

为了减少文件大小，删除以下目录：

```
❌ .venv/                   # 虚拟环境（接收者会重建）
❌ .venv-gpu/               # GPU虚拟环境
❌ __pycache__/             # Python 缓存
❌ .git/                    # Git 历史（可选）
❌ .pytest_cache/           # 测试缓存
❌ models/                  # 模型目录（很大！接收者会自动下载）
❌ .claude/                 # 代码助手配置
❌ *.log                    # 日志文件
❌ *.pyc                    # 编译文件
❌ node_modules/            # Node 依赖（npm会重装）
```

## 📦 打包方式

### 方式 1：手动创建 ZIP 包（推荐）

**Windows 用户**：
1. 选中项目目录中除了上述排除文件外的所有文件
2. 右键 → 发送到 → 压缩文件夹
3. 重命名为 `RAG_系统.zip`

**Mac/Linux 用户**：
```bash
# 进入项目的父目录
cd /path/to/
zip -r RAG_系统.zip 终极代码/ \
  -x "终极代码/.venv/*" \
  -x "终极代码/__pycache__/*" \
  -x "终极代码/models/*" \
  -x "终极代码/.git/*" \
  -x "终极代码/*.log" \
  -x "终极代码/node_modules/*"
```

### 方式 2：Git 方式

如果你的项目在 GitHub：

```bash
git clone https://github.com/你的用户名/RAG系统.git
```

确保 `.gitignore` 包含了要排除的文件：

```
# .gitignore
.venv/
.venv-gpu/
__pycache__/
models/
*.log
node_modules/
.pytest_cache/
```

### 方式 3：云盘分享

- 上传到 **OneDrive** / **Google Drive** / **百度网盘**
- 更新方便
- 接收者可以随时获取最新版本

## 💡 额外建议

### 1. 提供需求文档
创建 `系统需求.txt`：
```
系统要求：
- Python 3.8+
- 20 GB 磁盘空间
- 8+ GB 内存（推荐 16GB）
- 网络连接
- LLM API 账号（讯飞/OpenAI/DeepSeek 等）

可选：
- NVIDIA 显卡（用于加速）
- Node.js（仅用于前端开发）
```

### 2. 提供示例 .env 文件
让接收者知道需要填什么：
```
# config/.env.example
LLM_API_KEY=填入你的_API_密钥_此处
LLM_BASE_URL=https://spark-api-open.xf-yun.com/v2/
LLM_MODEL=spark-x

# 其他 LLM 选项：
# OpenAI: https://api.openai.com/v1
# DeepSeek: https://api.deepseek.com
```

### 3. 提供常见问题 FAQ
```
Q: 装 Python 时应该装哪个版本？
A: 3.11 或 3.10，需要勾选 Add to PATH

Q: 安装要多久？
A: 10-15 分钟，取决于网络

Q: 没有 API 密钥能用吗?
A: 不能，必须有一个 LLM 服务的 API

Q: 需要显卡吗？
A: 不需要，但有的话会更快
```
