# 📖 新用户快速启动指南

**给代码接收者的完整操作说明**

---

## 👋 你好！收到了 RAG 智能问答系统的代码

本指南会一步步教你如何快速运行这个程序。不需要编程知识，按照步骤操作即可。

**预计耗时**: 10-15 分钟（不包括模型下载时间）

## ✅ 前置检查清单

在开始之前，请确保你的电脑具备以下条件：

### 必需条件
- [ ] **Windows 10+ 或 Mac/Linux**
- [ ] **互联网连接**（用于下载依赖）
- [ ] **至少 20 GB 空闲磁盘空间**（用于模型和数据库）
- [ ] **至少 8 GB 内存**（推荐 16 GB）

### 可选条件
- [ ] **NVIDIA 显卡**（提速 Embedding，但不必需）
- [ ] **Node.js 已安装**（仅为运行前端需要）

## 🚀 第 1 步：安装 Python（如果还没装）

### Windows 用户

1. 访问 https://www.python.org/downloads/
2. 下载 **Python 3.11**（推荐的稳定版本）
3. 运行下载的安装程序
4. **重要**：勾选 **"Add Python to PATH"** ✅

![Python 安装screenshot]
```
□ Install launcher for all users
□ Add Python 3.11 to PATH              ← 必须勾选！
```

5. 点击 **Install Now** 完成安装
6. 打开 PowerShell 验证：
   ```bash
   python --version
   ```
   应该显示 `Python 3.11.x`

### Mac 用户

```bash
# 使用 Homebrew
brew install python@3.11
```

### Linux 用户（Ubuntu/Debian）

```bash
sudo apt-get install python3.11 python3.11-venv python3-pip
```

---

## 🚀 第 2 步：解压代码

1. 将收到的压缩包解压到一个方便的位置
   - 例如：`C:\项目\RAG系统` 或 `~/RAG系统`
2. **记住这个路径**，后面会用到

```
解压后的目录结构应该像这样：
终极代码/
├── install.py           ← 一键安装脚本
├── install.bat          ← Windows 安装脚本
├── doctor.py            ← 诊断工具
├── quickstart.py        ← 启动菜单
├── run.bat              ← Windows 菜单
├── config/
│   └── .env.example     ← 需要重命名为 .env
├── data/
├── src/
├── frontend/
└── requirements.txt
```

---

## 🚀 第 3 步：一键安装所有依赖

### Windows 用户（最简单方法）

1. 打开文件管理器，进入代码目录
2. **找到 `install.bat` 文件**
3. **双击运行** `install.bat`
4. 等待安装完成（会打开一个黑色窗口，显示安装进度）

**或者使用 PowerShell**：
```powershell
cd C:\你的项目路径\终极代码
python install.py
```

### Mac/Linux 用户

打开终端，运行：
```bash
cd 你的项目路径/终极代码
python install.py
```

### 安装脚本会自动做什么？

安装脚本 `install.py` 或 `install.bat` 会：

✅ 检查 Python 版本  
✅ 检测 GPU（如有）  
✅ 创建必要的目录  
✅ **生成 `.env` 配置文件**（重要！）  
✅ 下载并安装所有 Python 库  
✅ 下载并安装 Node.js 依赖（如有 Node.js）  
✅ 构建前端应用  
✅ 验证安装成功  

**安装完成后会显示**：
```
✓ 成功的检查: 15
⚠ 警告: 0
✗ 错误: 0

安装成功！
```

> ⏱️ **耗时**: 5-10 分钟（取决于网络速度）

## 🚀 第 4 步：配置 API 密钥（最重要！）

安装完成后，你需要配置 LLM API 密钥才能让系统正常工作。

### 打开配置文件

**Windows**：
- 在代码目录找到 `config` 文件夹
- 找到 `.env` 文件（可能显示为 `~.env`）
- 右键 → 打开方式 → 记事本

**Mac/Linux**：
```bash
nano config/.env
```

### 填写 API 密钥

找到以下几行，根据你选择的 LLM 服务填写：

#### 选项 A：使用讯飞星火（推荐入门）

```env
LLM_API_KEY=你的_API_密钥
LLM_BASE_URL=https://spark-api-open.xf-yun.com/v2/
LLM_MODEL=spark-x
```

**获取方式**：
1. 访问 https://console.xf-yun.com/
2. 注册账号
3. 复制你的 API Key
4. 粘贴到上面的 `你的_API_密钥` 位置

#### 选项 B：使用 OpenAI
```env
LLM_API_KEY=sk-xxxxxxxxxxxxx
LLM_BASE_URL=https://api.openai.com/v1
LLM_MODEL=gpt-4
```

#### 选项 C：使用 DeepSeek
```env
LLM_API_KEY=sk-xxxxxxxxxxxxx
LLM_BASE_URL=https://api.deepseek.com
LLM_MODEL=deepseek-chat
```

### 保存文件

- **Windows**：Ctrl + S
- **Mac/Linux**：Ctrl + X → Y → Enter

✅ **完成！现在系统知道用哪个 LLM 了**

## 🆘 遇到问题了？

### 问题 1：显示 "Python 不存在"

**解决**：
1. 重新安装 Python（记住勾选 Add to PATH）
2. 重启电脑
3. 重新运行 install.py

### 问题 2：显示 "缺少某个模块"

**解决**：
```bash
python install.py    # 重新运行安装脚本
```

### 问题 3：前端无法显示

**解决**：
1. 确保已安装 Node.js（访问 https://nodejs.org/）
2. 重新运行 install.py
3. 再次启动

### 问题 4：API 认证失败

**解决**：
1. 检查 `config/.env` 文件
2. 确保 `LLM_API_KEY` 填写正确
3. 保存文件
4. 重新启动应用

### 问题 5：不确定系统状态

**解决**：运行诊断工具
```bash
python doctor.py
```

这会显示详细的系统信息和问题诊断。
