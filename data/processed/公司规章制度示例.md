# 示例规章制度文档

## 第一章 总则

**第一条** 为规范公司管理，建立良好的工作秩序，根据国家相关法律法规，结合公司实际情况，特制定本规章制度。

**第二条** 本制度适用于公司全体员工，包括正式员工、试用期员工和实习生。劳务派遣人员参照本制度执行。

**第三条** 公司倡导"以人为本、诚信务实、创新进取"的企业文化，全体员工应当遵守职业道德，维护公司利益和形象。

## 第二章 考勤管理

**第四条** 工作时间

1. 标准工作时间：周一至周五，上午9:00至下午6:00
2. 午休时间：12:00至13:00（不计入工作时间）
3. 弹性工作时间：经部门主管批准，可在8:00-10:00之间灵活到岗，相应调整下班时间

**第五条** 考勤打卡

1. 员工须在规定工作时间前后15分钟内完成打卡
2. 每月迟到超过3次者，按每次50元扣发绩效奖金
3. 因特殊原因无法打卡的，需在当天向人事部门提交书面说明

**第六条** 请假制度

1. **年假**：入职满一年后，可享受5-15天带薪年假（根据工龄递增）
2. **病假**：凭医院证明可申请病假，年度累计不超过30天
3. **事假**：需提前3个工作日申请，年度累计不超过15天
4. **婚假**：正式员工结婚可享受3天带薪婚假
5. **产假**：女性员工享受98天产假，符合政策可延长

**第七条** 请假审批流程

| 请假天数 | 审批人 | 备案 |
|---------|-------|------|
| 1天以内 | 直属主管 | 无需 |
| 2-3天 | 部门经理 | 人事备案 |
| 3天以上 | 总经理 | 人事备案 |

## 第三章 薪酬福利

**第八条** 薪酬构成

员工薪酬由以下部分组成：
1. 基本工资：根据岗位和职级确定
2. 绩效奖金：根据季度绩效考核结果发放
3. 年终奖：根据公司年度盈利情况和个人绩效发放
4. 津贴补贴：包括餐补、交通补贴、通讯补贴等

**第九条** 发薪时间

1. 每月10日发放上月工资（遇节假日顺延）
2. 绩效奖金按季度发放，时间为每季度结束后次月15日
3. 年终奖于春节前发放

**第十条** 福利待遇

1. **五险一金**：公司依法为员工缴纳养老保险、医疗保险、失业保险、工伤保险、生育保险和住房公积金
2. **补充医疗保险**：公司为入职满一年的员工购买商业医疗保险
3. **节日福利**：端午、中秋、春节等传统节日发放节日礼品或慰问金
4. **生日福利**：员工生日当月享受200元生日礼金

## 第四章 行为规范

**第十一条** 职业道德

1. 诚实守信，不得弄虚作假
2. 保守公司商业秘密和客户信息
3. 不得利用职务之便谋取私利
4. 维护公司形象，不得散布有损公司声誉的言论

**第十二条** 办公礼仪

1. 着装整洁得体，符合职业形象
2. 接待来访应热情礼貌
3. 办公区域保持安静，不大声喧哗
4. 爱护办公设备，节约用水用电

**第十三条** 信息安全

1. 妥善保管账号密码，定期更换
2. 不得将公司电脑用于私人用途
3. 未经批准不得将公司数据外传或拷贝
4. 发现安全隐患应及时报告

## 第五章 奖惩制度

**第十四条** 奖励

对有以下表现的员工，公司将给予表彰和奖励：
1. 工作业绩突出，超额完成目标的
2. 提出合理化建议被采纳并产生效益的
3. 在突发事件中表现突出，避免公司损失的
4. 获得重要职业资格证书或荣誉称号的

**第十五条** 处罚

对违反公司规章制度的员工，视情节轻重给予以下处罚：
1. **警告**：轻微违规，口头或书面警告
2. **记过**：一般违规，扣除当月绩效奖金的20%
3. **降职降薪**：严重违规，调整岗位并降低薪资级别
4. **解除劳动合同**：重大违规，依法解除劳动合同

**第十六条** 申诉渠道

员工对处罚决定有异议的，可在收到处罚通知后5个工作日内向人事部门提出书面申诉。人事部门应在10个工作日内组织复核并给予答复。

## 第六章 附则

**第十七条** 本制度由人事部门负责解释和修订。

**第十八条** 本制度自发布之日起施行，原有相关制度同时废止。

**第十九条** 公司保留对本制度的最终解释权。
