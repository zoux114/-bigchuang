# RAG 系统安装工具总结

## 📦 已创建的工具

### 1️⃣ **install.py** (Python 一键安装脚本)
   - 文件路径: `终极代码/install.py`
   - 功能: 
     * 检查 Python 版本和模块
     * 检测 GPU 支持
     * 创建目录结构
     * 安装 Python 和 Node.js 依赖
     * 构建前端
   - 使用: `python install.py`
   - 特点: 跨平台（Windows/Mac/Linux）

### 2️⃣ **install.bat** (Windows 一键安装脚本)
   - 文件路径: `终极代码/install.bat`
   - 功能: 完全相同，但针对 Windows 优化
   - 使用: 双击运行或 `install.bat`
   - 特点: 无需打开 PowerShell，更新手

### 3️⃣ **doctor.py** (系统诊断工具)
   - 文件路径: `终极代码/doctor.py`
   - 功能:
     * 检查 Python 版本
     * 验证所有关键模块
     * 检测 GPU 并显示详细信息
     * 检查目录和文件完整性
     * 验证配置文件
     * 检查 Node.js/npm
     * 显示系统资源（内存、磁盘）
   - 使用: `python doctor.py`
   - 何时使用: 安装后验证或遇到问题时诊断

### 4️⃣ **quickstart.py** (Python 交互式菜单)
   - 文件路径: `终极代码/quickstart.py`
   - 功能: 提供交互式菜单，快速访问所有操作
   - 菜单选项:
     1. 一键安装
     2. 诊断系统
     3. 初始化向量数据库
     4. 启动 API 服务器
     5. 启动前端开发服务器
     6. 启动完整应用
     7. 打开项目文件夹
     8. 编辑环境配置
     9. 查看帮助文档
   - 使用: `python quickstart.py`
   - 特点: 更友好的用户界面

### 5️⃣ **run.bat** (Windows 交互式菜单)
   - 文件路径: `终极代码/run.bat`
   - 功能: 同 quickstart.py，但针对 Windows
   - 使用: 双击运行或 `run.bat`
   - 特点: 不需要 Python 命令行

### 📄 **INSTALL.md** (详细安装指南)
   - 文件路径: `终极代码/INSTALL.md`
   - 内容:
     * 系统要求
     * 快速安装步骤
     * 详细的分步说骤
     * 安装内容说明
     * 常见问题解答
     * 高级配置选项

### 📄 **使用工具参考.md** (本文件)
   - 文件路径: `终极代码/使用工具参考.md`
   - 内容: 工具概览、使用指南、故障排除

## 🚀 快速开始

### 首次使用流程（3步）

```bash

```

```bash

# 然后选择想要的操作
```

## 🛠️ 绑定配置

### 环境变量 (config/.env)

```env
# LLM 配置（必需）
LLM_API_KEY=your_key_here
LLM_BASE_URL=https://spark-api-open.xf-yun.com/v2/
LLM_MODEL=spark-x

# Embedding 配置
EMBEDDING_MODEL=shibing624/text2vec-base-chinese  
EMBEDDING_DEVICE=auto

# 其他配置
VECTOR_DB_DIR=data/vector_db
LOG_LEVEL=INFO
```

### 支持的 LLM

```
讯飞星火（默认）
├─ BASE_URL: https://spark-api-open.xf-yun.com/v2/
├─ MODEL: spark-x / spark-4.0-ultra
└─ 需要: API_KEY

OpenAI
├─ BASE_URL: https://api.openai.com/v1
├─ MODEL: gpt-4 / gpt-3.5-turbo
└─ 需要: API_KEY

DeepSeek
├─ BASE_URL: https://api.deepseek.com
├─ MODEL: deepseek-chat
└─ 需要: API_KEY

智谱
├─ BASE_URL: https://open.bigmodel.cn/api/paas/v4
├─ MODEL: glm-4 / glm-3-turbo
└─ 需要: API_KEY
```
