## 🚀 使用指南

### 最简单的使用方式（3 步）

#### Windows 用户
```bash
# 步骤 1: 运行一键安装
install.bat

# 步骤 2: 编辑配置（在菜单中选择 8 或手动编辑）
编辑 config\.env

# 步骤 3: 启动应用（在菜单中选择 6）
run.bat 或 python quickstart.py
```

#### Mac/Linux 用户
```bash
# 步骤 1: 运行一键安装
python install.py

# 步骤 2: 编辑配置
nano config/.env

# 步骤 3: 启动应用
python quickstart.py
```

### 完整的工作流

```
1. 首次安装
   └─ python install.py (或 install.bat)

2. 验证环境
   └─ python doctor.py

3. 配置 API
   └─ 编辑 config/.env

4. 准备数据
   └─ 将文件放入 data/raw/

5. 初始化数据库
   └─ python quickstart.py → 选择 3

6. 启动服务
   └─ python quickstart.py → 选择 6

7. 访问应用
   └─ http://localhost:8000 或 8001
```

### 如果安装失败

```bash

python doctor.py

# 步骤 2: 查看错误日志
# （install.py 会显示详细错误）

# 步骤 3: 重新尝试
python install.py

# 步骤 4: 手动修复
# 编辑 config/settings.py
```

### 清除和重置

```bash
# 删除虚拟环境（如有）
rmdir /s /q venv

# 删除缓存
rmdir /s /q models/__pycache__

# 重新安装
python install.py
```
