# ⚡ 别人需要做什么才能运行这个程序？

**简明版本 - 只需 3 步**

---

## 📋 接收者的完整待办清单

### ✅ 第 1 步：安装 Python（5 分钟）

**如果电脑上没有 Python，需要安装一个：**

```
访问 → https://www.python.org/downloads/
下载 → Python 3.11 或 3.10
安装 → 选中 "Add Python to PATH" ✅
```

验证安装：
```bash
python --version
```

---

### ✅ 第 2 步：运行安装程序（10 分钟）

接收者收到代码后，这一步最重要！

#### Windows 用户

进入代码目录，双击 **`install.bat`** 或运行：
```bash
python install.py
```

#### Mac/Linux 用户

```bash
python install.py
```

**这个脚本会自动做的事**：
- ✅ 检查系统
- ✅ 安装所有依赖
- ✅ 创建配置文件
- ✅ 下载模型

---

### ✅ 第 3 步：配置 API 密钥（2 分钟）

编辑 `config/.env` 文件，填入 API Key：

**选择一个 LLM 服务并填写**：

```env
# 讯飞星火
LLM_API_KEY=你的API密钥
LLM_BASE_URL=https://spark-api-open.xf-yun.com/v2/
LLM_MODEL=spark-x

# 或 OpenAI
# LLM_API_KEY=sk-xxxxx
# LLM_BASE_URL=https://api.openai.com/v1
# LLM_MODEL=gpt-4
```

---

### ✅ 第 4 步：启动应用（1 分钟）

```bash
python quickstart.py
# 或 Windows: run.bat
```

选择菜单选项 **6**

然后打开浏览器访问：
```
http://localhost:8000
```

## 🚨 如果出错了？

### 最常见的问题

**"Python 不是内部或外部命令"**
→ 重装 Python，勾选 Add to PATH

**"模块缺失"**
→ 再运行一次 `python install.py`

**"API 认证失败"**
→ 检查 `config/.env` 中的 API Key 是否正确

**"不知道哪里出错"**
→ 运行 `python doctor.py` 诊断
