  ▶ 第 1 步：一键安装

    Windows 用户：
      方法 A: 双击 install.bat
      方法 B: python install.py

    Mac/Linux 用户：
      python install.py

    ⏱️ 预计时间: 5-10 分钟

  ▶ 第 2 步：配置 API 密钥

    打开 config/.env 文件，找到以下行：

      LLM_API_KEY=your_api_key_here
      LLM_BASE_URL=...
      LLM_MODEL=...

    填入你的 API 密钥和配置

  ▶ 第 3 步：启动应用

    Windows 用户：
      run.bat          # 选择选项 6
      或
      python quickstart.py  # 选择选项 6

    Mac/Linux 用户：
      python quickstart.py  # 选择选项 6

    然后访问：http://localhost:8000
