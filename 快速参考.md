## 🆘 快速故障排除

| 问题 | 解决方案 |
|------|--------|
| Python 找不到 | 下载 Python 3.8+ 并勾选 "Add to PATH" |
| 模块缺失 | 运行 `python install.py` |
| GPU 不可用 | 装 CUDA 12.1+ 和显卡驱动，然后 `python install.py` |
| 前端无法访问 | 装 Node.js，运行 `python install.py` |
| API 认证失败 | 检查 `config/.env` 中的 API_KEY |
| 诊断命令有问题 | 运行 `python doctor.py` 查看详情 |

## ⚙️ 环境变量完整配置

```env
# === LLM 配置 (必需) ===
LLM_API_KEY=your_key_here
LLM_BASE_URL=https://spark-api-open.xf-yun.com/v2/
LLM_MODEL=spark-x
LLM_TEMPERATURE=0.7
LLM_MAX_TOKENS=2048

# === Embedding 模型 ===
EMBEDDING_MODEL=shibing624/text2vec-base-chinese
EMBEDDING_DEVICE=auto  # 或 'cuda' / 'cpu'
EMBEDDING_BATCH_SIZE=32

# === 数据库 ===
VECTOR_DB_DIR=data/vector_db
VECTOR_DB_INDEX_TYPE=Flat

# === 分块 ===
CHUNK_SIZE=500
CHUNK_OVERLAP=50

# === 其他 ===
LOG_LEVEL=INFO
PROJECT_ROOT=.
```

## 🎓 核心概念

```
RAG (Retrieval-Augmented Generation)
  │
  ├─ Retrieval (检索)
  │  ├─ Embedding (向量化)
  │  ├─ Vector DB (向量数据库)
  │  └─ Similarity Search (相似度搜索)
  │
  └─ Generation (生成)
     ├─ Query Processing
     ├─ Context Retrieval
     └─ LLM Response
```
